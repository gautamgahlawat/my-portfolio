About

Enthusiastic Engineer!

I'm a Front-End/ Fullstack web developer based in Delhi, India.
My passion is building incredible user experiences - optimizing frontend and backend applications to deliver fast, responsive, consistent and performant UX to all users. I've also done brochures, logos, posters, pitch decks, and more.

Love New Technologies!

I am a passionate engineer with an immense interest and love in Javascript development. I thrive in environments that embrace new technologies. I don't hesitate to try out new things if it is made to ease the workflow and make development faster and consistent across all devices.

Contact Me


Recipe of this site

This site is created using HTML, CSS, Bootstrap and JavaScript;Carefully hand-coded in Sublime Text; compiled by Grunt; hosted on Github; The codebase is version-controlled by Git and managed on a public Github account.The type is set in beautiful Montserrat and Lato served by Google Fonts. The photographs were taken as screenshots and svg avatar from Hexatar.

Note for visitors !!!

This site is designed to be future-friendly, and I have tried to make it backward compatible but still forgive me if it doesn’t play or work well on your older browsers. As time goes on, I may roll out patches to improve support but no guarantees 😎. It’s also a playground for me so I’m constantly trying new stuff, making changes, and probably breaking things.
